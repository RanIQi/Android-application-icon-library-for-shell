Without a computer, I searched countless methods but couldn't find a direct way to get the application icon.  
aapt2's implementation is complex, and I failed to successfully convert XML to PNG.  
I kept wondering why such a useful tool as the shell couldn't get the application icon?  
Even though no one seemed to care, I stepped in.  
The main idea is to use the shell to download the GitHub icon to the local machine,  
and then use the local machine to retrieve the application icon.
